Grid

The client wanted to aggregate all the images on social media platforms such as Twitter and Instagram with the tag #Marc, #MarcFam, or #Marc Jacobs. My team designed the viewer for all these aggregated images -- we chose a six column grid with endless scrolling.

Hover Zoom

For this project we wanted the user to be able to preview the images without having to download them or opening them in another link. We chose to have the images that are viewable to expand and show their full size upon the user hovering the cursor over it.

Single Image Viewer

For each image, we had to have a screen to allow the user to view the image, share it, or view the owner. We designed a single image viewer on the users click.
